SERIAL PORTS OF 8051

It is so called because we are transmitting the bits serially.
The pins TXD and RXD are used to transmit and receive the bits.
Each bit requires 1 cycle to transmitt and receive.
so, 8-bits require 8 cycles to transmit and receive.
Inorder to save the processor's cycles, an 8-bit register called SBUF is used.
The processor sends the 8-bit data to SBUF in 1 cycle and the SBUF transmitts bit by bit in 8 cycles
After transmitting or receiving the DATA SBUF will interrupt the PROCESSOR with the help of Ti and Ri .
The flags Ti and Ri are Autocleared.
If an error is present in data. Dta present in SBUF is cleared.

⭐️ SCON
SCON - Serial Control (8-bit) (BItwise)

SM0 SM1 SM2 REN TB8 RB8 Ti Ri
  • Ti,Ri - Transmitter and receiver interrupts
    • Ti,Ri are automatocally set after the data is sent or received to inform the processor.
    • Ti,Ri are automatocally cleared after going to ISR.
  • REN - Receiver Enable
    • REN = 1 (Receiver is Enabled)
    • REN = 0 (Reciver is Disabled)
  • TB8 - TX 9th BIT
    • Holds the the 9th bit of the Transmitter .
    • Used only in M2,M3 .
  • RB8 - RX 9th BIT
    • Holds the the 9th bit of the Receiver .
    • Used only in M1,M2,M3 .
  • SM2
    • used by the programmer to do error checking or not.
    • SM2 = 1 (Error checking is done on the Received data)
    • SM2 = 0 (Error checking is not done on the Received data)
  • SM0,SM1 - used to select modes

1.M0 (shift register) :

d8
--------
  • All 8-bits are sent and received normally.
  • This mode does not do Error Cheching.
  • It simply accepts the data.
  • In this mode RXD is 1/2Duplex and TXD is used to send CLK.
  • This mode is not used in UART

M1,M2,M3 are FULL DUPLEX

2.M1 (8-bit UART) :

START d8 stop
0 -------- 1
  • Here start and stop bits are system generated but Microprocessor sends only 8-bits.
  • This mode has 1 Start bit and 1 Stop bit.
  • The stop bit is used to do Error checking in this mode.
  • Start bit is aleays 0 , to wake the PROCESSOR.
  • Stop bit should always be 1 to end the data.
    • if stop bit = 1 data is accepted.
    • if stop bit = 0 Error in the programmer.

3.M2,M3 (8-bit UART) :

Start d8 9th BIT Stop
0 -------- 0/1 1
  • 9th bit is given by the programmer.
  • Error checking is done by the 9th bit.
    • If the 9th bit = 1 data is accepted.
    • If the 9th bit = 0 error in data.
